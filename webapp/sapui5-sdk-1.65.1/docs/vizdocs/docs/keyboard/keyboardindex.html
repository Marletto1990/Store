[TAB]

Forward Navigation:

  • On Enter, first time: focus moves to the first visible data point of the first series. The first series is the one, which is written on top in the legend.
  • On Enter, any consecutive time: Move focus to the item which had the focus before.
  • If focus is on Item level: Move focus to the Category / Time Axis, which corresponds to the formerly focused data point (if interactive), otherwise to the Legend.
  • If focus is on Category / Time Axis: The first time, focus moves to the first legend entry. Any consecutive time, focus moves to the legend entry, which had the focus before.
  • If focus is on Legend: Leave the chart.

[SHIFT] + [TAB]

Backward Navigation:

  • On Enter, first time: Move out of the chart.
  • If focus is on Legend: Move Focus to the Category / Time Axis (if interactive), which had the focus before. (At first time: Move focus to the first visible Category / Time Axis). If Category / Time Axis is not interactive, move focus to Item level.
  • If focus is on Category / Time Axis: Move focus to the one which had the focus before. (at first time: Move focus to the first visible one).
  • If focus is on Item level: Leave the chart.

[F6]

Forward Navigation:

No matter where the focus resides within the Chart, move it to the first element in the tab chain of the next F6-group. Depend on UI5 to move focus to charts.


[SHIFT] + [F6]

Backward Navigation:

No matter where the focus resides within the Chart, move it to the first element in the tab chain of the previous F6-group. Depend on UI5 to move focus to charts.


Arrow Keys

On Item level:

[Up] / [Down]

  • For horizontal chart, navigate always to the previous / next data point in the same series, even if there is a gap in between.
  • For vertical chart, navigate always to the previous / next data point in the same catetory, even if there is a gap in between.
  • For Treemap, navigate always to the closest top/bottom left data point or category label which shares the same border.
  • Do not cycle. Stop at the last / first item.

[Left] / [Right]

  • For horizontal chart, navigate always to the previous / next data point in the same catetory, even if there is a gap in between.
  • For vertical chart, navigate always to the previous / next data point in the same series, even if there is a gap in between.
  • For Treemap, navigate always to the closest top left/right data point or category label which shares the same border.
  • Do not cycle. Stop at the last / first item.

On Category / Time Axis:

  • Move to the next / previous one. On one level, [LEFT] equals [UP], [RIGHT] equals [DOWN]; On multi-level, all arrow keys act independently.
  • Do not cycle. Stop at the last / first one.

On Legend Level:

  • Move to the next / previous legend item. [LEFT] equals [UP], [RIGHT] equals [DOWN].
  • Do not cycle. Stop at the last / first legend item.

Note:

  • If needed, scroll the chart to keep the focus item visible.
  • Bar / Dual Bar / Column / Dual Colum / Bullet is an exception, in which [Up] equals to [Left], [Down] equals to [Right], and cannot navigate by series.
  • In Combination, [Left] / [Right] in bar means navigating to the previous / next data point in the same Category / Time Axis, but [Left] / [Right] in line means navigating to the previous / next data point in the same series. Press [Up] / [Down] to navigate from line to bar(or bar to line), it will find the nearest data point to navigate.
  • In Bubble / Scatter, press [Up] / [Down] / [Left] / [Right] will navigate to the nearest data point in the direction.
  • In Pie / Donut, [Up] = [Left] (anticlockwise), [Right] = [Down] (clockwise).

[SPACE] / [ENTER]

On Item level:

Select / Deselect the current item. (Selection in addition to existing selection, depend on corresponding selection mode.)

On Category / Time Axis:

Select / Deselect the all items in the Category / Time Axis item. (Selection in addition to existing selection, depend on corresponding selection mode.)

On Legend Level:

Select / Deselect all items described by the Legend item. (Selection in addition to existing selection, depend on corresponding selection mode.)

Note:

  • Open the pop-over / tool-tip (or keep it open) and focus will move to it.

[ALT] + [UP]

On Item level:

[ALT] + [UP] moves the focus to the first data point of the previous series. The previous series is the one shown above the current series in the legend.If the first data point of the previous series is outside the visible area, scroll the chart so that the focus is inside the visible area. If focus is on the first series, [ALT] + [UP] does nothing.


[ALT] + [DOWN]

On Item level:

moves the focus to the first data point of the next series. The next series is the one shown below the current series in the legend. If the first data point of the next series is outside the visible area, scroll the chart so that the focus is inside the visible area.If focus it on the last series, [ALT] + [DOWN] does nothing.


[Plus] + [Minus]

If zooming is enabled, [Plus] will zoom in and [Minus] will zoom out. Only applicable in plot area and Category Axis.
