Retrieving Files From a Single SAPUI5 Repository

You can retrieve the files (also the conflicting files) from your project.

  1. From the context menu of your project, choose Team  Retrieve.
  2. To also select the conflicting files, choose Select All.
  3. Choose Finish.
Note

If you use special characters (such as Chinese, Japanese, or Cyrillic), you must ensure that they are supported by the code page of your project. You can define this on project level, but we recommend that you do it on workspace level as the team provider could overwrite the project settings during downloading. Choose Window  Preferences  General  Workspace in Eclipse and under Text file encoding and Other: select UTF-8.

(On project level, you'd right-click your project and choose Project  Properties  Resource from the context menu and define UTF-8 as text file encoding.)
