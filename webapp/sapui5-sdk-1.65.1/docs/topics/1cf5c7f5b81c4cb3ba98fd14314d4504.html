List Report Elements

The list report allows the user to work with a large list of items. It combines powerful functions for filtering large lists with different ways of displaying the resulting item list.

NoteThe following image contains links to more information.

Main Elements of the List Report

The list report view includes the following main elements:

  • Application header

  • Smart filter bar with variant management and the generic Share menu that includes the following actions:

    • Send Email

    • Save as Tile
    • Share in SAP Jam (if integration with SAP Jam is configured)
    As shown above, the smart filter bar is displayed by default when users launch an app. When you choose Go, the content of the list report is displayed.

    If users have set Execute on Select for their default variant, the content of the list report is displayed immediately when the app is launched; the smart filter bar is collapsed.

  • Smart table

  • Footer toolbar that can include optional actions
