SAPUI5 Tools for Eclipse

The SAPUI5 repository team provider enables you to share, submit, and retrieve content of single SAPUI5 repositories with the respective projects in Eclipse.

Prerequisites

  • You have installed the SAPUI5 app development feature and the SAPUI5 repository team provider in your Eclipse installation.

    For more information, see Installing SAPUI5 Tools for Eclipse.

  • The ABAP system you are using must be available in your SAP GUI logon group. We recommend that you enable single sign-on.
Note

We recommend to use SAP Web IDE for developing complex apps with the latest innovations. SAPUI5 tools for Eclipse are only suitable for simple use cases.
